# Pocket_StrMod
A deep learning model enabling rapid bioactivity optimization

### Package Requirements:
All the requirements are in requirements.txt, you can use 
~~~shell
pip install -r requirements.txt
~~~

### Command for optimization
The molecule can be generated by running the following command, where the pocket pdb file and the model parameter file are required, and the rest of the parameters are optional
~~~shell
python main_generate.py -pkt pockets/Mpro_initial/Mpro_initial_pocket15.pdb -lig pockets/Mpro_initial/scaffold.sdf --idx "[12, 14, 15, 16]" --ckpt ./ckpt/150000.pt -n 500 --name Mpro_optimization --root_path gen_results -d cuda:0 -at 1.0 -bt 1.0 --max_atom_num 28 -ft 0.5 -cm True --with_print False
~~~
All parameters of generation:
~~~

### molecular number
To specify the growing sites of the ligand, using molecule_number.py to show the number of atoms.
~~~python
from rdkit import Chem
from rdkit.Chem import AllChem
from rdkit.Chem import  Draw
from rdkit.Chem.Draw.MolDrawing import  DrawingOptions 

def show_atom_number(mol, label):
    for atom in mol.GetAtoms():
        atom.SetProp(label, str(atom.GetIdx()))
    return mol

opts = DrawingOptions()
opts.includeAtomNumbers=True
m = Chem.MolFromMolFile("XXX.sdf", sanitize=1) 
AllChem.Compute2DCoords(m)
show_atom_number(m, 'atomLabel')
opts.includeAtomNumbers=True
opts.bondLineWidth=2.8
image = Draw.MolToImage(m, options = opts, size=(500,500))
image.save("XXX.png")
~~~


### Spliting Pocket
Based on the pose of the ligand, the pocket structure can be splited from the protein structure
~~~python
from pocket_flow import SplitPocket, Protein, Ligand

pro = Protein('/path/to/protein.pdb')
lig = Ligand('/path/to/ligand.sdf')
dist_cutoff = 10
pocket_block, _ = SplitPocket._split_pocket_with_surface_atoms(pro, lig, dist_cutoff)
open('/path/to/pocket.pdb','w').write(pocket_block)
~~~

### fine-tuning Dataset
The raw [CrossDocked2020]dataset could be downloaded here: (https://bits.csb.pitt.edu/files/crossdock2020/) 
The program to process the dataset is:
~~~python
from pocket_flow import CrossDocked2020

unexpected_sample = [
    line.split()[-1] for line in open('data/unexcept_element_sample_new.csv').read().split('\n')
    ]
cs2020 = CrossDocked2020(
    './data/crossdocked_pocket10/',
    './data/crossdocked_pocket10/index.pkl',
    unexpected_sample=unexpected_sample
    )
cs2020.run(
    dataset_name='crossdocked_pocket10_processed_35Atoms.lmdb',
    max_ligand_atom=35,
    only_backbone=False,
    lmdb_path='./data/'
    )
~~~
